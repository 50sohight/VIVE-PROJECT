# AI English assistant
Первый проект сделанный на 1 курсе входе проектного практикума. Данный чат-бот на телеграмме дает пользователю возмонжость изучать английский в 3 режимах: плановый, словарь, свободный.

## Плановый режим
Пользователю дается возможность выбрать свой уровень языка от чего ему предлагается план для обучения, построенный на роликах по темам и тестов после них.

## Словарь режим
Пользователю взависимости от его уровня языка предлагается выбрать перевод для определенного слова, после чего производиться проверка на корректнность.

## Свободный режим
Пользователю предлагается общаться с ИИ(ChatGPT) на английском, а после завершения диалога выводиться подробное описание ошибок человека допущенных при общении.

В данном проекте мной были написаны плановый режим и сама архитектура чат-бота.
